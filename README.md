# Curse of Zed — Title Screen (Java Swing)

## Requirements
- Java JDK 11 or higher (NetBeans already includes this)
- Ringbearer Medium font (free — see below)

---

## Font Setup (IMPORTANT)
1. Go to: https://www.dafont.com/ringbearer.font
2. Click Download, unzip
3. Copy RingbearerMedium.ttf into the CurseOfZed project root
   (same folder as background.jpg, next to src/)

If the font is missing the program still runs with a fallback font.

---

## Run in NetBeans
1. File > New Project > Java > Java Application > Next
2. Name it CurseOfZed > Finish
3. Delete the auto-generated Main.java
4. Copy src/CurseOfZed.java into your project src/ folder
5. Copy background.jpg and RingbearerMedium.ttf into the project ROOT
   (folder containing src/, build/, nbproject/)
6. Right-click CurseOfZed.java > Run File  (or Shift+F6)

---

## Run via Command Line
Windows:   run.bat
Mac/Linux: ./run.sh
